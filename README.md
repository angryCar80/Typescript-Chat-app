# Typescript Chat App

a simple chat app in typescript that uses bun

To install dependencies (There is no dependencies):

```bash
bun install
```

To run:
To run the app you should run the server first

```bash
bun run server.ts
```

and then run the client file

```bash
bun run client.ts
```

multiple clients is allowed and works great but i never tested a lot of user (more then 10+)
